## 🚀 Что делать дальше:

### 1. Выполни SQL в Supabase (2 минуты)

1. Открой **Supabase Dashboard**
2. Перейди в **SQL Editor**
3. Создай **New Query**
4. Скопируй весь код из `supabase-lobby-schema.sql`
5. Нажми **Run** (Ctrl+Enter)
6. Дождись успешного выполнения ✅

### 2. Запусти приложение (30 секунд)

```bash
npm run dev
```

### 3. Протестируй систему лобби (5 минут)

#### Тест 1: Создание лобби и выбор персонажа
1. Открой http://localhost:5173
2. Введи имя (любое)
3. Нажми **"Создать лобби"**
4. Увидишь экран выбора персонажа
5. Выбери одного из 4 тестовых персонажей:
   - Торин (Дварф-воин)
   - Элара (Эльф-следопыт)
   - Гримм (Человек-плут)
   - Мирабель (Полурослик-клерик)
6. Попадешь в комнату ожидания

#### Тест 2: Создание своего персонажа
1. Нажми **"+ Создать персонажа"**
2. Заполни форму:
   - Имя (уникальное!)
   - Раса (выбери из списка)
   - Класс (выбери из списка)
   - Предыстория (опционально)
3. Нажми **"Создать"**
4. Автоматически присоединишься к лобби

#### Тест 3: Присоединение второго игрока
1. Открой **новую вкладку** (или инкогнито)
2. Скопируй код лобби (например, ABC123)
3. Введи другое имя
4. Введи код лобби
5. Нажми **"Войти"**
6. Выбери **ДРУГОГО** персонажа (Торин будет 🔒 Занят)
7. Попадешь в ту же комнату ожидания
8. Увидишь обоих игроков!

#### Тест 4: Начало игры
1. Оба игрока нажимают **"Готов к игре"**
2. Хост (первый игрок) нажимает **"Начать игру"**
3. Оба переходят в чат
4. AI видит обоих персонажей!

## 🎮 Как это работает:

```
Главное меню
    ↓
Создать лобби / Войти в лобби
    ↓
Выбор персонажа
    ├─ Выбрать существующего
    └─ Создать нового
    ↓
Комната ожидания
    ├─ Видны все участники
    ├─ Кнопка "Готов"
    └─ Хост может начать игру
    ↓
Игра (Chat)
    └─ Все персонажи в одной комнате
```

## 📊 Проверка в Supabase:

После тестов можешь проверить данные:

```sql
-- Все персонажи
SELECT * FROM characters;

-- Активные лобби
SELECT * FROM lobbies WHERE is_active = true;

-- Участники лобби
SELECT 
  lp.*,
  c.name as character_name,
  c.race,
  c.class
FROM lobby_participants lp
JOIN characters c ON lp.character_id = c.id;
```

## 🐛 Если что-то не работает:

### Ошибка "таблица не найдена"
→ Выполни `supabase-lobby-schema.sql` в Supabase SQL Editor

### Персонаж не освобождается
→ Вручную очисти:
```sql
DELETE FROM lobby_participants WHERE lobby_id = 'YOUR_LOBBY_ID';
```

### Не видно других игроков
→ Проверь, что оба в одном лобби (одинаковый код)

### Кнопка "Начать игру" неактивна
→ Все игроки должны нажать "Готов к игре"
