## 🎯 Что это?

Система многоуровневой обработки AI-запросов для RPG-приложений:

```
┌──────────────┐
│ Пользователь │
└──────┬───────┘
       │
       ▼
┌──────────────────────────────────────────────────────────┐
│ AI Orchestrator                                          │
│                                                          │
│  1. Проверяет историю (>15 сообщений?)                 │
│  2. Суммаризирует старые (Gemini Flash - бесплатно)    │
│  3. Берет последние 10 сообщений                        │
│  4. Формирует оптимальный контекст                      │
└──────┬───────────────────────────────────────────────────┘
       │
       ▼
┌──────────────────────────────────────────────────────────┐
│ OpenRouter → Claude 3.5 Sonnet / GPT-4                  │
│ Генерирует креативный ответ DM                          │
└──────┬───────────────────────────────────────────────────┘
       │
       ▼
┌──────────────┐
│ Ответ игроку │
└──────────────┘
```

## 🎯 Возможности

```
✅ Многоуровневая архитектура
   └─ Gemini Flash для суммаризации (бесплатно)
   └─ Claude/GPT-4 для ответов (качество)

✅ Оптимизация токенов
   └─ Суммаризация после 15+ сообщений
   └─ Хранение последних 10 сообщений
   └─ Кэширование сводок

✅ Retry логика
   └─ Автоматические повторы при ошибках
   └─ Exponential backoff (1s → 2s → 4s)
   └─ Обработка 429/500/502/503

✅ Гибкая конфигурация
   └─ 8 готовых конфигураций
   └─ Выбор модели
   └─ Настройка параметров

✅ Полное тестирование

   └─ Сравнение стоимости
```
