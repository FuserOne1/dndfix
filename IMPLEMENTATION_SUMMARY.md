## 🎯 Ключевые возможности

### 1. Многоуровневая архитектура
```
User → Orchestrator → Gemini Flash (суммаризация)
                   → OpenRouter (Claude/GPT-4)
                   → Response
```

### 2. Оптимизация токенов
- Суммаризация после 15+ сообщений
- Хранение последних 10 сообщений
- Кэширование сводок
- **Экономия: ~57%**

### 3. Retry логика
- Автоматические повторы при ошибках
- Exponential backoff (1s → 2s → 4s)
- Обработка 429/500/502/503

### 4. Гибкая конфигурация
- 8 готовых конфигураций
- Выбор модели
- Настройка параметров
- Динамическое переключение

### 5. Полное тестирование
- Веб-интерфейс
- Консольные тесты
- Сравнение стоимости
- Проверка работы

## 🔧 Технические детали

### Архитектура
- **Frontend:** React 19, TypeScript
- **AI Layer:** OpenRouter + Gemini
- **Database:** Supabase
- **Build:** Vite

### Модели
- **Основная:** Claude 3.5 Sonnet / GPT-4 / Gemini Pro
- **Суммаризация:** Gemini Flash 1.5 (бесплатно)

### API
- **OpenRouter:** Unified LLM API
- **Gemini:** Бесплатно до 1500 запросов/день
- **Supabase:** Realtime + PostgreSQL
