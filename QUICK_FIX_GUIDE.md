# 🔧 Быстрое исправление - Инструкция

## Проблема: Персонажи не удаляются

### Решение (1 минута)

1. Откройте ваш проект в Supabase
2. Перейдите в **SQL Editor**
3. Скопируйте и выполните этот код:

```sql
DROP POLICY IF EXISTS "Anyone can delete characters" ON characters;

CREATE POLICY "Anyone can delete characters" ON characters
  FOR DELETE USING (true);
```

4. Нажмите **Run** (или F5)
5. Готово! Теперь персонажи удаляются ✅

## Новая функция: Сохранения

### Как продолжить путешествие

1. На главном экране найдите **"Сохраненные путешествия"**
2. Выберите нужное сохранение
3. Нажмите на него
4. Игра продолжится! 🎮

### Что показывает сохранение

- 📜 Код комнаты
- 👤 Имя персонажа
- ⏰ Когда играли (например, "2 ч назад")
