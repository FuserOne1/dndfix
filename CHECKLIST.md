# ✅ Checklist - Внедрение AI Orchestrator

## 📋 Подготовка (5 минут)

- [ ] Зарегистрируйся на [OpenRouter](https://openrouter.ai/)
- [ ] Пополни баланс OpenRouter ($5-10)
- [ ] Получи API ключ OpenRouter
- [ ] Получи API ключ [Google Gemini](https://ai.google.dev/) (бесплатно)
- [ ] Скопируй `.env.example` в `.env`
- [ ] Заполни ключи в `.env`

## 🔧 Настройка (10 минут)

- [ ] Открой `src/components/Chat.tsx`
- [ ] Добавь импорт: `import { AIOrchestrator } from '../lib/ai-orchestrator';`
- [ ] Создай ref: `const orchestratorRef = useRef<AIOrchestrator | null>(null);`
- [ ] Инициализируй в `useEffect` (см. MIGRATION.md)
- [ ] Замени вызов Groq на `orchestratorRef.current.processMessage()`
- [ ] Сохрани файл

- [ ] Проверь, что AI отвечает
- [ ] Отправь еще 20 сообщений

## 🐛 Troubleshooting

### Ошибка: "API key invalid"
- [ ] Проверь `.env` файл
- [ ] Убедись, что ключи правильные
- [ ] Перезапусти `npm run dev`

### Ошибка: "Rate limit exceeded"
- [ ] OpenRouter: проверь баланс
- [ ] Gemini: подожди 1 минуту (лимит 15/мин)

### Ответы не в контексте
- [ ] Проверь, что суммаризация работает
- [ ] Увеличь `RECENT_HISTORY_COUNT` до 15
- [ ] Уменьши `SUMMARIZE_THRESHOLD` до 10

### Медленные ответы
- [ ] Нормально для первых запросов
- [ ] Попробуй модель `openai/gpt-4-turbo` (быстрее)
- [ ] Уменьши `maxTokens` до 512
